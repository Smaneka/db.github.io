Simpan Data ke GitHub

Form untuk menyimpan data input ke repository GitHub

Cara Setup:

  1. Buat personal access token di GitHub dengan permissions repo
  2. Ganti nilai variabel GITHUB_TOKEN dan GITHUB_USERNAME di JavaScript
  3. Pastikan repository yang dituju sudah ada di akun GitHub Anda
  4. Token akan terlihat seperti: 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
